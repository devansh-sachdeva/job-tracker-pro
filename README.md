# JobTracker Pro 🚀

A full-stack web application to track job and internship applications.

## 🌐 Live Demo
https://job-tracker-pro-4ya8.onrender.com

## ✨ Features
- User Authentication (JWT + bcryptjs)
- Track applications — Applied → Interview → Offer → Rejected
- Real-time dashboard with statistics
- Live search and filter by company/status
- Secure HTTP-only cookie sessions

## 🛠️ Tech Stack
- **Backend:** Node.js, Express.js
- **Database:** Supabase (PostgreSQL)
- **Auth:** JWT + bcryptjs
- **Frontend:** EJS Templates, HTML, CSS
- **Deployed:** Render

## 🚀 Getting Started
- Clone the repo
- Run `npm install`
- Add `.env` file with Supabase credentials
- Run `node index.js`
